## Project Overview
This project showcases an interactive **Excel dashboard** built using the Sample Superstore dataset.  
The goal is to analyze sales and profit performance across regions, categories, sub-categories, and time.

The dashboard is designed for **business stakeholders** to quickly identify trends, top-performing segments, and problem areas.

## Dashboard Features
The Excel dashboard includes:

- **Sales by Category**
  - Compares Furniture, Office Supplies, and Technology
- **Sales by Region**
  - Central, East, South, West performance comparison
- **Profit by Sub-Category**
  - Identifies high-profit and loss-making products
- **Monthly Sales Trend**
  - Time-series analysis from 2014 to 2017
- **Interactive Year Slicer**
  - Allows filtering all visuals by year
